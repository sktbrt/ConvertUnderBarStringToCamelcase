# ConvertUnderBarStringToCamelcase
アンダーバーで区切られた文字列をキャメルケースに変換します。


| Before        | After          |
| ------------- |:--------------:|
| a |A |
| aaa |Aaa |
| a_aaa |AAaa |
| a_aa_aaa |AAaAaa |
| ab_cde |AbCde |
| ABCD |Abcd |
| AbCdEfG |Abcdefg |
| Abc_Defg |AbcDefg |
| a_a_a_A_B_bbb_Bbb |AAAABBbbBbb |
| _aaa |Aaa |
